Quick start
===========

Compare configurations
----------------------

Step 1: Run benchmarks
~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: console

   make benchmarks

Step 2: Run the fmax synthesis
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. tabs::

   .. group-tab:: Vivado

      .. code-block:: console

         make vivado

   .. group-tab:: Design Compiler

      .. code-block:: console

         make dc

Step 3: Visualize and explore the results
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: console

   make explore

Run simulations
---------------

.. code-block:: console

   make sim
